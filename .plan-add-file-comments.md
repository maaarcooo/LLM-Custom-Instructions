# Plan: Add Full Name Comments to Prompt Files

## Objective
Add a comment at the top of each prompt/instruction text file indicating its full identifier name, making it easy to identify the file's purpose when viewed in isolation.

## Comment Format
Use `# ` prefix (already used in existing files) with the full name:
```
# anki-flashcard-prompt-v4
```

## Implementation
Each comment will be added as the **first line** of the file, followed by a blank line for readability.

**Example (prompt-v4.txt before):**
```
<prompt>
Using this PDF as a reference...
```

**Example (prompt-v4.txt after):**
```

<prompt>

```
